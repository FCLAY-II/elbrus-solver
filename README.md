## Elbrus Bootcamp

Сервис предназначен для людей, которые только начинают свой путь в изучение JavaScript, пользователь может зарегистрироваться/авторизоваться и переходить к решению задач, он будет оповещен когда решит верно или нет ту или иную задачу, а также введется статистика пользователя.
